# 1:1 Meeting Prep

Generate a comprehensive 1:1 meeting prep document for the specified employee.

**Arguments:**
- `<employee-name>` — the name of the team member (required)

**Process:**

1. Read `~/.boss-ai-agent/data/employees.json` and find the employee by name.
2. If employee not found, suggest closest match or list all employees.
3. Read the employee's data from the last 30 days:
   - Report files from `~/.boss-ai-agent/data/reports/`
   - Signal files from `~/.boss-ai-agent/data/signals/`
4. If GitHub is enabled: WebFetch the employee's recent PRs and commits.
5. Generate a structured prep document:
   - **Performance Overview**: submission rate, trend (improving/declining/stable)
   - **Sentiment Trend**: mood trajectory over 30 days
   - **Recent Blockers**: from reports and channel messages
   - **Code/Task Contributions**: from GitHub (if available)
   - **Suggested Topics**: 3-5 data-driven discussion topics
   - **Conversation Strategy**: mentor-specific advice for the 1:1
6. Present the document to the boss.
